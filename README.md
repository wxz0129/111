# 📊 Investment Research Auto-Sorter (研报自动化整理工具)

## 📖 项目简介 (Introduction)

这是一个专为金融从业者设计的 Python 自动化脚本，用于解决大量卖方研报（PDF/Excel）的文件命名混乱和分类归档问题。

它能够自动扫描 `Investment Research` 文件夹中的文件，通过**文件名正则匹配**和**PDF内容深度扫描**，识别公司或行业，并将其重命名为标准格式后移动到对应的目录结构中。

**核心痛点解决：**
- 告别手动重命名（自动清洗为 `券商-公司-标题-日期` 格式）。
- 自动区分“个股报告”与“行业策略”。
- 自动归档模型文件（Excel）至独立子文件夹。
- **完全便携**：基于相对路径开发，无需修改代码路径，即插即用。

---

## 🚀 功能特性 (Features)

1.  **智能重命名**：
    - 格式：`Broker-Company-Title-YYMMDD.ext`
    - 自动清洗无用的 Ticker 后缀、特殊字符。
2.  **多维度识别逻辑**：
    - 优先级：Excel 关键词 > PDF 文件名 Ticker > 文件名公司全称 > **PDF 文本内容扫描** (OCR-like) > 行业关键词。
3.  **自动目录构建**：
    - 个股：`Output/公司名/报告` 和 `Output/公司名/模型`
    - 行业：`Output/行业报告`
    - 未分类：`Output/_未分类文件` (便于人工二次检查)
4.  **防冲突机制**：
    - 针对短代码（如 "T", "U", "SE"）设有特殊保护逻辑，防止误匹配。
